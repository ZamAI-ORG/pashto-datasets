# Pashto Dataset: Creating Dataset

Welcome to the **Pashto Dataset Creating Dataset** repository!

This repository is dedicated to the collection, curation, and sharing of high-quality datasets for the Pashto language. Our goal is to accelerate research, development, and educational projects involving Pashto Natural Language Processing (NLP).

## About

This project aims to create and maintain open datasets for the Pashto language, supporting tasks such as language modeling, translation, speech recognition, and more. The dataset is meant to empower researchers, developers, and enthusiasts working on Pashto language technologies.

## Dataset Structure

The repository may include:
- **Text corpora** (news articles, literature, social media, etc.)
- **Parallel corpora** (Pashto-English or other language pairs)
- **Speech data** (audio files and transcriptions)
- **Metadata** (source, license, description)

Example directory structure:
data/ ├── text/ ├── parallel/ ├── speech/ └── metadata/
