# Pashto Text Dataset Visualizations

This directory contains visualizations of the Pashto text dataset processing pipeline:

## 1. Complete Data Pipeline
![Complete Pashto Data Pipeline](/asset/pashto_complete_data_pipeline.png)

This visualization shows the complete end-to-end pipeline for Pashto text data processing, including data sources, extraction processes, cleaning and normalization steps, and the final output formats.

## 2. Data Processing Workflow
![Pashto Data Processing Workflow](/asset/pashto_data_processing_workflow.png)

This visualization shows the logical flow between different data sources and processing scripts in the Pashto text dataset workflow.

## 3. Text Transformation Process
![Pashto Text Transformation Process](/asset/pashto_data_transformation_process.png)

This visualization shows the step-by-step transformation of Pashto text from raw data to fine-tuning ready format.

## How These Visualizations Were Created

The visualizations were generated using Python with Matplotlib. The source code for generating these visualizations can be found in the scripts directory:

- `/scripts/generate_workflow_visualization.py`
- `/scripts/generate_transformation_visualization.py`
- `/scripts/generate_complete_pipeline_visualization.py`

You can regenerate these visualizations by running the respective scripts.
